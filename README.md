 # 小海博客后端管理系统

  

  ## 基于Springboot的后端博客管理系统




  ### 主要使用的技术

|    使用的技术    |    名称    |     版本      |
| :--------------: | :--------: | :-----------: |
|     后端框架     | Springoot  | 2.1.3.RELEASE |
|       orm        |  mybatis   |     2.0.1     |
|       分页       | pageHelper |    1.2.12     |
|     权限管理     |   shiro    |     1.3.2     |
|     项目构建     |   Maven    |     3.6.1     |
|     运行环境     |   Ubuntu   |  16.04.6 LTS  |
|     接口文档     |  Swagger   |     2.6.1     |
|   数据库连接池   |   druid    |    1.1.14     |
| 缓存（线上环境） |   redis    |     3.0.6     |
|数据库|mysql|5.7|

  ### 接口文档

   项目采用swagger2，接口文档自动生成，具体为 http://ip:端口/swagger-ui.html
